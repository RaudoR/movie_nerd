Features

Bellow you will see a demo of the main 3 features of MovieNerd.

Search by Title

When you type the movie title, you will get all the objects (movies) associated to your input.

Movie Details

With the Movie Details feature, you will see specific information like: cast, release date, genre... etc for whatever movie you choose.

IMDB ID

This feature allows you check the official IMDB page for any of the movies, in case you want more information on where to watch the movie for example.
